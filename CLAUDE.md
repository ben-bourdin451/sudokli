## Project

Sudokli — a terminal-based sudoku puzzle game built with Rust, using ratatui for the TUI and crossterm for terminal input. Early stage: currently renders an empty 9x9 grid with 3x3 box shading. Uses Rust edition 2024.

## Commands

- **Build**: `cargo build`
- **Run**: `cargo run`
- **Check (fast compile check)**: `cargo check`
- **Tests**: `cargo test`
- **Lint**: `cargo clippy`
- **Format**: `cargo fmt` (check only: `cargo fmt -- --check`)
